Class PFValidateResult

  • java.lang.Object
    • net.phonefactor.pfsdk.PFValidateResult
public class PFValidateResult
extends java.lang.Object
Class used for returning the result of a PFAuth.validateDeviceToken call. This class tells you if the validation was successful or not, and the status of validation. Generally speaking, you only need to check if validation was successful.
See Also:
