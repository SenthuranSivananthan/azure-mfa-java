The PhoneFactor Java SDK.

The basic method for using the PhoneFactor SDK is to create a PFAuth instance, initialize it via the initialize method, optionally change the instance's default behavior via the various setter methods, and then call the authenticate method on the instance. Use the PFAuthResult instance returned from the PFAuth.authenticate call to determine succes, failure, and other details about what transpired. It is recommended that you create, initialize, and customize one PFAuth instance (say at application initilization) and use that one instance across all of your application threads. The call to PFAuth.authenticate() and PFAuth.userSync() is thread safe.

Consult the file Example.java (included with the SDK) for a working example implementation of authentication. Consult the file ExampleUserSync.java (included with the SDK) for a working example implementation of user synchronization.
