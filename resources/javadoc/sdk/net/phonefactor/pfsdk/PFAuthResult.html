net.phonefactor.pfsdk

Class PFAuthResult

  • java.lang.Object
    • net.phonefactor.pfsdk.PFAuthResult
public class PFAuthResult
extends java.lang.Object
Class used for returning the result of a PFAuth.authenticate call. This class tells you if the authentication was successful or not, the status of the call that was placed, and any message errors received. Generally speaking, you only need to check if authentication was successful. You may check the other results such as call status to determine other actions to take.
See Also:
net.phonefactor.pfsdk.PFAuth.authenticate
