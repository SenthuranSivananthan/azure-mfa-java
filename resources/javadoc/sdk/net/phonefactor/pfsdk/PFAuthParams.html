PFAuthParams

public PFAuthParams(String username,
                    String countryCode,
                    String phoneNumber,
                    String extension,
                    String backupCountryCode,
                    String backupPhoneNumber,
                    String backupExtension,
                    String language,
                    AuthModeInfo authInfo,
                    EventInfo eventInfo)
Creates a PFAuthParams populated with values
Parameters:
username - The username to authenticate
countryCode - The country code (required) -- ex: "1" for the United States
phoneNumber - The phone number to dial
extension - The extension to dial once answered -- set to null for no extension (can contain "," for a pause, "*", or "#")
backupCountryCode - The country code for the backup phone number
backupPhoneNumber - The phone number to try if the primary phone fails
backupExtension - The extension to dial when the backup phone is answered
language - The language to use for the authentication. This is primarily used for selecting the correct voice message for phone authentications. Defaults to "en" if set to null.
authInfo - Information about the authentication mode of this request. Set to null if you want to use standard phone authentication (press # to authenticate). Otherwise, you can pass in an instance of SmsInfo, StandardPinInfo, PhoneAppInfo, etc.
eventInfo - Any information related to the even this auth represents. Pass in null if there is no event associated with this auth. Contact a PhoneFactor representative for more information on events.
