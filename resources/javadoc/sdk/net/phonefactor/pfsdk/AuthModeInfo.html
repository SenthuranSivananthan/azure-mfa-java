public abstract class AuthModeInfo
extends java.lang.Object
Abstract base class for all possible authentication modes. An authentication mode could be pin, sms, phone app, or others. You should never instantiate this class. Rather, instantiate one of the derived classes such as StandardPinInfo or SmsInfo.
