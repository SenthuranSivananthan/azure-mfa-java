SHA1PinInfo

public SHA1PinInfo(String hexHash,
                   String salt)
Creates a SHA1PinInfo with the provided HEX STRING representation of the binary hash data and plaintext salt. This method is provided as a convenience when pulling info from a database or other datasource that has already provided you with the binary data encoded as a string of hex digits.
Parameters:
hexHash - The SHA1 hash encoded as a string of hex digits
salt - The plaintext salt

SHA1PinInfo

public SHA1PinInfo(int[] hash,
                   String salt)
Creates a SHA1PinInfo with the provided binary hash data and plaintext salt.
Parameters:
hash - The SHA1 hash -- each int in this array should contain an 8 bit unsigned value (0-255) representing an octet of hash data.
salt - The plaintext salt
