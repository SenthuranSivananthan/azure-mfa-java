public abstract class PinInfo
extends AuthModeInfo
Abstract base for all PinInfo classes. PinInfo derived classes allow you to pass different types of already hashed pins to the authenticate method of PFAuth. You should create a new instance of one of the derived classes as an argument to PFAuth.authenticate() depending on what type of pin hashing was used (if any). You should never create this class directly.
