SmsInfo

SmsInfo class constructor.
Parameters:
smsModeIn - The integer constant representing the mode of this sms auth (otp or otp-pin).
smsDirIn - The integer constant representing the direction of this sms auth (one-way or two-way).
pinFormatIn - The integer constant representing the format of the pin for this sms auth (pos1, sha1, or plainText). Use only when using SMS_MODE_OTP_PIN.
pinIn - The pin for this authentication.
